## Visão Geral
Projeto de desenvolvimento de um website voltado ao público externo para o restaurante **Estância d'Oliveira**. O objetivo da plataforma é facilitar o acesso à informação para os clientes, automatizar os pedidos e criar um canal de comunicação para contratações.

## Funcionalidades Principais

O sistema contará com as seguintes funcionalidades:

* **Cardápio e Preços:** Listagem atualizada dos valores das refeições correspondentes a cada dia da semana.
* **Sistema de Pedidos:** Área de cadastro e login de usuários para a solicitação e gerenciamento de pedidos de marmitas.
* **Trabalhe Conosco:** Formulário dedicado para que os visitantes possam submeter seus currículos caso tenham interesse em integrar a equipe do restaurante.
